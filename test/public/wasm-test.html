WASM Configuration

Expected WASM File Locations:

  • /wasm/web-ifc.wasm
  • /wasm/web-ifc-mt.wasm

Required MIME Types:

  • application/wasm for .wasm files

Common Issues & Solutions

File Not Found Errors:

  • Make sure the WASM files are in your public/wasm/ directory
  • Run npm run setup-wasm to copy the files from node_modules/web-ifc/dist/

MIME Type Errors:

  • For IIS servers, make sure web.config is in your public/wasm/ directory
  • For Apache servers, make sure .htaccess is in your public/wasm/ directory
  • For development servers, check if your dev server is configured to serve WASM files correctly

Manual Check

You can manually check if the WASM files are accessible by clicking these links:

web-ifc.wasm web-ifc-mt.wasm

Note: The browser might download these files instead of displaying them.
