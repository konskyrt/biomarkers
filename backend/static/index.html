BIM Data Dashboard API

This is the backend API for the BIM Data Dashboard application.

Available Endpoints:

/api/materialauszug/upload (POST) - Upload IFC files
/api/materialauszug/files (GET) - List uploaded IFC files
/api/materialauszug/file/:file_id (GET) - Get a specific IFC file
/api/bim2log/process (POST) - Process BIM data with timeline
